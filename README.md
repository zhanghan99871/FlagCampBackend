# Initialization

to build the project, you may need to:

1. `git clone https://github.com/zhanghan99871/FlagCampBackend.git`
2. `cd FlagCampBackend`
3. `docker compose up -d db`
4. For Mac or Linux:
   `chmod +x gradlew`
   `./gradlew build`
   For Windows:
   `gradlew.bat build`
5. `./gradlew bootRun`
